# git-setting-enable README

## Features

Quickly enable and shut down git

## Usage

1、（Ctrl + Shift + P / Cmd + Shift + P）
2、 select `Toggle Git Enabled`



## Extension Settings

* `myExtension.enable`: Enable/disable this extension.
